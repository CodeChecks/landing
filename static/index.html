Building a frontend app? Just use Visual Regression codecheck to find visual changes with laser precision. For backend development, you may want to warn review about breaking changes in API based on GraphQL or Swagger definitions. Measure test coverage, build size, type coverage, whatever you want.  There’s a ton of possibilities and everything is configured directly in your code.

Powerful API

A codecheck is a piece of JavaScript code that can run any logic and access Code Checks’ API. Starting from inspecting files changed in a given PR to running sophisticated visual regression on screenshots from base branch — everything is possible. Our API helps you to store and interact with artifacts and notify GitHub about results. Just imagine all the possibilities!
